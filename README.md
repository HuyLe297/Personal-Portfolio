# Personal Portfolio

A modern, responsive personal portfolio website built with **React** and **TailwindCSS**. Showcasing my skills as a Front-End Developer, with smooth scrolling, hover effects, and a clean dark theme. Deployed on Vercel for fast performance.

[Live Demo](https://personal-portfolio-swart-delta.vercel.app/)

## Features

- **Responsive Design**: Perfect on mobile, tablet, and desktop (TailwindCSS utilities).
- **Smooth Scrolling Navigation**: One-page layout with anchor links and smooth scroll behavior.
- **Skills & Tools Showcase**: Interactive grid with icons (HTML, CSS, JavaScript, React, TailwindCSS, Laravel, VS Code, Git, Figma, etc.) – hover effects for engagement.
- **Projects Section**: Featured projects with cards, images, descriptions, and links to Live Demo & GitHub.
- **Contact Form**: Functional form powered by [Formspree](https://formspree.io) – sends messages directly to email.
- **Dark Theme**: Sleek, professional dark mode with gradients and shadows.
- **Performance Optimized**: Built with Vite for fast builds and loading.

## Sections Overview

- **Hero**: Introduction with name, profession, and CTA to view work.
- **About**: Personal bio as a passionate Front-End Developer focusing on React and TailwindCSS.
- **Skills**: Two grids – "My Skills" (core tech) and "My Tools" (dev tools) with hover animations.
- **Projects**: 3 featured projects (JobFinder AI, Weather App, Personal Portfolio) with images and links.
- **Contact**: Form + social links (GitHub, LinkedIn, Email) for easy outreach.
